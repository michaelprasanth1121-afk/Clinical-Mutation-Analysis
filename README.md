# Clinical Mutation Analysis using SAS

This project demonstrates an end-to-end Clinical SAS workflow using colorectal cancer clinical data.

## Project Workflow

* Raw Excel Data Import
* QC Analysis
* Data Cleaning
* SDTM Domain Creation

  * DM
  * MH
  * TU
  * CE
* ADaM Dataset Development
* TLF Generation
* PDF Report Export

## Technologies Used

* SAS
* PROC SQL
* PROC FREQ
* PROC MEANS
* PROC SGPLOT
* ODS PDF

## Key Features

* SDTM Mapping
* ADaM Dataset Creation
* Mutation Analysis
* Statistical Summaries
* Clinical Reporting

## Output

Final PDF report generated using ODS PDF.
